CS:GO Esports Database


SQL database server for professional CS:GO players, teams, and events

Python (pandas, BeautifulSoup, Selenium, pymssql, SQLAlchemy), SQL, Docker, SSMS, Visio, Ubuntu, Proxmox

  • Created entity relationship diagrams to plan and visualize the various elements of the database and their interactions (relations) using Visio, as well as the data pipelines that represented the process from getting the data to storing in the database server.
  • Used pandas, Beautifulsoup4 and Selenium to scrape professional CS:GO players’- and teams’ performance statistics from HLTV.org into pandas dataframes.
  • Created an Ubuntu-based Linux container template on a self-hosted Proxmox server to host a Microsoft SQL Server image inside a Docker container; and used the pymssql and SQLAlchemy libraries in Python to connect to the database and send the pandas dataframes to the database server.
  • Managed the database using Microsoft SQL Studio Management Studio (SSMS).

Store Branches Sales Regression Analysis


Regression analysis performed on a stores branches sales dataset

R (R Markdown, base R, lm(), psych, RColorBrewer, dplyr)

  • Performed exploratory analysis on the dataset using summary statistics (mean, median, standard deviation, skew, etc.), central tendencies (quartiles).
  • Created a variety of linear regression models by applying various transformations (log, power, etc.) in order to optimize the model.
  • Addressed issues of multi-collinearity by calculating Variance Inflation Factors (VIF) and dropping correlated predictor variables.
  • Improved the R-squared by 34.25% in the final model from the starting model. Analyzed and addressed issues in the dataset that could have resulted in weak model such as weak predictors.
  • Consolidated the models analyses in an R Markdown document.
